This is a work in progress of a plugin created to facilitate easy footer link management.

== Description ==

Sets up a way to add special footer links where wordpress calls wp_footer() - can be used for things like adding tracking codes.

== Installation ==

1. Upload whole wc-footer-links folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
